Conwayn game of life.

Säännöt:
1. Solu muuttuu eläväksi, jos sen naapureista tasan 2 tai 3 on eläviä
2. Solu pysyy elävänä, jos tasan 3 naapuria on eläviä
3. Muussa tapauksessa solu kuolee

Toteutus:
1. Pelilautaa kuvataan matriisilla
2. Matriisialkioista 0 kuvaa kuollutta ja 1 elävää solua
3. Elävän solun tarkistus tapahtuu summaamalla ympäröivien solujen arvot

Grafiikka:
1. Kirjastona PyGame
2. Solujen arvot kuvataan eri väreillä
